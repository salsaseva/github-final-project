# Simple Interest Calculator

A script that calculates simple interest given principal, annual rate of interest and time period in years.

## Usage

```bash
./simple-interest.sh
```

## Input
- `p`: principal amount
- `t`: time period in years
- `r`: annual rate of interest

## Output
- `simple interest = p*t*r`
